In this folder:

SCRIPTS:
- entropy_plots.ipynb
	Jupyter notebook for plotting/analyzing results 
- get_entropy_values.py
	python script for functions fo rgenerating entropy values per subject and overall
- use_get_entropy_vals.py
	use functions in get_entropy_values 


INPUTS TO SCRIPTS (videos not in GitHub since they're ginormous)
- DOWNTOWN DAY.mp4/DOWNTOWN NIGHT.mp4
	original video files for downtown
- DOWNTOWN DAY-entropy.mp4/DOWNTOWN NIGHT-entropy.mp4
	entropy versions of downtown videos
- DOWNTOWN_DAY_resized.avi/DOWNTOWN_NIGHT_resized.avi
	original videos resized to match entropy vids, for input into write_fix_vid_double
- DOWNTOWN_DAY_grayscale.mp4/DOWNTOWN_DAY_grayscale.mp4
	grayscale version of videos
- downtown_day_max_entropy.csv/downtown_night_max_entropy.csv
	max entropy values for each frame for downtown videos (generated in MATLAB)


OUTPUTS OF SCRIPTS:
- day_ave_entropy.p/night_ave_entropy.p
	python pickle object containing average entropy values per each frame in full videos (generated by get_ave_entropy() in get_entropy_values.py
- day_ave_normed_entropy.csv/night_ave_normed_entropy.csv
	NORMALIZED average entropy for first 1000 frames (data used in normed_ave_entropy_per_1000.png) get_ave_entropy def in get_entropy_values (before i changed it to be not normalized)
- day_min_vals.p/night_min_vals_300.p
	pickle of array of smallest entropy val per each frame (takes awhile to run so I only did first 300 values for night)
- overall_entropy_kde.png
	kernel density estimate plot of day vs. night entropy distributions
- p03_duration_kde.png
	kde plot of length of fixation distributions for day vs. night
- p03_ent_vs_duration.png
	entropy vs. duration for day and night
- p03_entropy_kde.png
	kde plot of entropy vals of day vs. night
- pilot03_day_entropy_df.csv/pilot03_night_entropy_df.csv
	csv of entropy/start time/end time/duration for each fixation (generated by get_entropy_values())
- pilot03_day_fix.csv/pilot03_night_fix.csv
	csv generated from MATLAB containing fixations
                channel labels:
                    - isFixation
                    - Original time
                    - New time (based on data collection srate)
                    - Frame count (which frame, aligns with original time)
                    - GIP x coordinate
                    - GIP y coordinate
- pilot03_day_fix.p/pilot03_night_fix.p
	pickle object containing data of csvs of same name, but formatted for input into get_entropy_values()
